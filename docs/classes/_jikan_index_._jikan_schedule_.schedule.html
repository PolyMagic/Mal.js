Class Schedule

Schedule

For more info visit https://jikan.docs.apiary.io

To get schedule you need to create schedule object, like so:

let schedule = Jikan.schedule();

And then you can use schedule object multiple times to get desired information.

schedule.all();
schedule.monday();
schedule.tuesday();
schedule.wednesday();
schedule.thursday();
schedule.friday();
schedule.saturday();
schedule.sunday();
schedule.other();
schedule.unknown();

Each of those functions returns promise

Example

 
Jikan.schedule().all()
              .then(res => res.data)
              .then(scheduleJson => {});
